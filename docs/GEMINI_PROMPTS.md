## 📋 Usage Instructions

### For Gemini (Google AI Studio or similar):

1. **Upload Reference Image First**
   - Go to Google AI Studio or Gemini interface
   - Upload `/public/images/og-image-3a.png` or `og-image-8b.png`

2. **Paste Prompt**
   - Copy one of the prompts above
   - Paste into the prompt field

3. **Generate**
   - Click generate
   - May need to generate 2-3 times to get the perfect version

4. **Adjust if Needed**
   - If text is wrong, add: "Keep exact text from reference image"
   - If layout is off, add: "Maintain exact layout and proportions from reference"
   - If too busy, add: "Reduce visual effects by 30%, keep it clean"

### Tips for Best Results:

**If Gemini changes the text:**
- Add at the start: "CRITICAL: Use exact text from reference image: [copy the actual text here]"
- Or generate without text, then overlay in Figma

**If layout changes too much:**
- Start prompt with: "This is a LAYOUT REFERENCE - maintain exact positioning and proportions"
- Emphasize: "Only enhance visuals, DO NOT change layout"

**If it's too busy/complex:**
- Add: "Keep enhancements subtle and professional"
- Or use the "Specific Enhancements" version with exact opacity values

**If colors are off:**
- Provide hex codes again
- Add: "Match colors exactly to reference image"
